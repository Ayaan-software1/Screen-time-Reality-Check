# Screen Time Reality Check

Log your daily screen time for 7 days — then see what that time could have been instead: book pages, walks, meals, hours outside. A small tool with a big guilt trip.

Single-file static app: vanilla JS + Tailwind (CDN), state in `localStorage`, German UI. No build step.

## Run it

Open `index.html` in a browser, or:

```
python -m http.server 8000
```
